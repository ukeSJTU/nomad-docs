#### 1. 功能概述 (Feature Overview)

本页定义了本系统承诺支持的软件和硬件环境，以确保绝大多数目标用户都能正常使用。

#### 2. 兼容性要求 (Compatibility Requirements)

| 项目             | 要求                                                                                             | 优先级          |
| :--------------- | :----------------------------------------------------------------------------------------------- | :-------------- |
| **桌面浏览器**   | - Google Chrome (最新版)<br />- Microsoft Edge (最新版)<br />- Firefox (最新版)<br />- Safari (最新版) | **Must Have**   |
| **移动端浏览器** | - Mobile Safari (iOS 最新版)<br />- Mobile Chrome (Android 最新版)                                 | **Should Have** |
| **操作系统**     | - 不限制桌面操作系统（Windows, macOS, Linux）。                                                  | **Must Have**   |
