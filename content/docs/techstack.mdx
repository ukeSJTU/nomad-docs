## 核心框架

### Next.js 15
全栈 React 框架，支持 Server Components、Server Actions 和现代化的 App Router

- **版本**: 15.5.4
- **特性**: App Router, Server Components, Turbopack (开发环境)

## 前端技术

### UI 组件库
- **Shadcn/UI**: 基于 Radix UI 的可复用组件库
- **Tailwind CSS 4**: 实用优先的 CSS 框架
- **Lucide React**: 现代化的图标库

### 主题与样式
- **next-themes**: 深色/浅色模式支持

## 后端与数据库

### 数据库
- **PostgreSQL**: 关系型数据库
- **Neon DB**: 无服务器 PostgreSQL 托管服务

### 数据验证
- **Zod**: TypeScript 优先的 schema 验证库

### API 层
- **Server Actions**: Next.js 15 原生的服务器端函数

## 开发工具

### 代码质量
- **ESLint**: JavaScript/TypeScript 代码检查
- **Prettier**: 代码格式化工具
- **TypeScript**: 类型安全的 JavaScript 超集

### CI/CD
- **GitHub Actions**: 自动化工作流和部署
- **Vercel**: 前端应用托管和部署平台

## 文档工具

### Fumadocs
基于 Next.js 的文档框架，支持 MDX 内容管理

- **fumadocs-ui**: UI 组件
- **fumadocs-core**: 核心功能
- **fumadocs-mdx**: MDX 内容处理
- **fumadocs-twoslash**: TypeScript 代码提示

### 内容增强
- **Mermaid**: 图表和流程图渲染
- **KaTeX**: 数学公式渲染
- **Shiki**: 代码语法高亮
